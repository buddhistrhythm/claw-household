POST 到下列 URL，默认请求体为 messages；鉴权改为「无」时可不配 Key（如 Ollama）。

需为合法 JSON；用 <<<MODEL>>>、<<<PROMPT>>> 插入模型名与已转义的用户提示词。

由服务端在 PATH 中执行命令，不经过 shell。参数为 JSON 数组，支持占位符 <<<PROMPT>>>、<<<MODEL>>>、<<<COUNT>>>、<<<URGENT>>>、<<<URGENT_DAYS>>>。取 stdout 第一行非空作为副标题。
